# QuantumMed
QuantumMed – Projekt (1) KrebsZellen-Killer

Quanten-Algorithmus entwickelt, der drei Krebs-Schlüsselproteine (E**, H**, K**) gleichzeitig blockt.

Real/Data. Echte Kristall-Strukturen von Krebs-Proteinen (aus der Protein-Datenbank, PDB)

Ziel: EGFR-Mutationen (bei Lungen-, Brust-, Darmkrebs).
Toxizität: Null

Herstellung: Struktur: 
SMILES-String=c1ccccc1C=The rest is Secret!

Wirkung: Krebszellen sterben in 48 h (Sim)

Details:  
- 32 Qubits, Qiskit + AerSimulator  
- Real-Kristallstrukturen aus PDB (1M17, 3PP0, 4L9W)
- Kohärenz 99.9 %, keine Fehler
